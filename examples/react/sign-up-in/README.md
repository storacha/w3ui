# Example

To run this example:

- Clone the w3ui repository and enter the `w3ui` directory

  ```sh
  git clone https://github.com/web3-storage/w3ui
  cd w3ui
  ```

- Install dependencies:

  ```sh
  pnpm install
  ```

- Change to this example directory and run the example:

  ```sh
  cd examples/react/sign-up-in
  pnpm run dev
  ```
